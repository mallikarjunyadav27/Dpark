DPARK.services.snow_auto.app.mlxtend.text.names.generalize_names(name, output_sep=' ', firstname_output_letters=1)[source]

Generalize a person’s first and last name.

Description : Returns a person’s name in the format <last_name><separator><firstname letter(s)> (all lowercase)

Parameters

  • name (str) – Name of the player

  • output_sep (str (default: ‘ ‘)) – String for separating last name and first name in the output.

  • firstname_output_letters (int) – Number of letters in the abbreviated first name.

Returns

gen_name – The generalized name.

Return type

str

Examples

For usage examples, please see http://rasbt.github.io/mlxtend/user_guide/text/generalize_names/

Generalizes names and removes duplicates.

Description : Applies mlxtend.text.generalize_names to a DataFrame with 1 first name letter by default and uses more first name letters if duplicates are detected.

Parameters

  • col_name (str) – Name of the DataFrame column where generalize_names function should be applied to.

Returns

df_new – New DataFrame object where generalize_names function has been applied without duplicates.
